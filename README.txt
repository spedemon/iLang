======
ilang
======

Imaging inference language. 
Typical usage looks like this::

    #!/usr/bin/env python

    from ilang import Graphs, Samplers, Tracers
    # ...



Installation
============

Linux, MacOsX
-------------

There are two ways to install under Linux and MacOsX: 

1. pip install ilang

2. python setup.py



Features
========

Samplers
--------

1. Metropolis Hastings Markov Chain Monte Carlo

2. Hamiltonian Monte Carlo 

3. Langevin Adjusted Metropolis Hastings Markov Chain Monte Carlo 


Optimizers
----------

3. Gradient Descent

4. Stochastic Gradient Descent

5. Expectation Maximization
